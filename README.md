# 图像拼接拆分工具

一个功能强大的图像处理工具，支持AI自动识别相似图片并拼接，以及将拼接图片拆分回独立图片。

## 功能特点

- **AI图像识别**：使用DINO模型自动识别同一物体的不同角度图片
- **图像拼接**：支持水平、垂直、网格三种拼接模式
- **图像拆分**：支持自动、水平、垂直、网格四种拆分模式
- **智能命名**：拆分后的图片自动按 "原名(序号)" 格式命名
- **图形界面**：简洁易用的GUI界面

## 安装

### 1. 安装Python依赖

```bash
cd image_tool
pip install -r requirements.txt
```

### 2. 首次运行

首次使用AI功能时，会自动下载DINO模型（约330MB），请确保网络连接正常。

## 使用方法

### 启动程序

```bash
python gui.py
```

### 图像拼接

1. 切换到"图像拼接"标签页
2. 点击"添加图片"选择要拼接的图片
3. （可选）点击"AI自动分组"自动识别相似图片
4. 选择拼接方式：
   - **horizontal**：水平拼接（从左到右）
   - **vertical**：垂直拼接（从上到下）
   - **grid**：网格拼接（需设置列数）
5. 设置输出目录和文件名
6. 点击"开始拼接"

### 图像拆分

1. 切换到"图像拆分"标签页
2. 点击"选择图片"选择要拆分的图片
3. 选择拆分方式：
   - **auto**：自动判断（根据图片宽高比）
   - **horizontal**：水平拆分
   - **vertical**：垂直拆分
   - **grid**：网格拆分（需设置行数和列数）
4. 设置输出目录和基础文件名
5. 点击"开始拆分"

### 命名规则

- 拼接后的图片命名为用户指定的名称（如 `stitched_image.png`）
- 拆分后的图片按 `基础名(序号).png` 格式命名
  - 例如：基础名为 `1`，拆分成3张图片
  - 输出：`1(1).png`, `1(2).png`, `1(3).png`

## AI分组功能

### 相似度阈值

- 默认值：0.85
- 范围：0.5 - 1.0
- 较高的值（如0.9）：更严格的相似度判断，减少误分组
- 较低的值（如0.7）：更宽松的相似度判断，可能将不同物体归为一组

### 工作原理

1. 使用DINO模型提取图片特征向量
2. 计算图片之间的余弦相似度
3. 使用DBSCAN聚类算法自动分组
4. 相似度超过阈值的图片将被归为同一组

## 技术栈

- **Python 3.8+**
- **Pillow**：图像处理
- **PyTorch**：深度学习框架
- **Transformers**：预训练模型
- **scikit-learn**：聚类算法
- **tkinter**：图形界面

## 注意事项

1. 首次运行AI功能需要下载模型，请耐心等待
2. 处理大量图片时可能占用较多内存
3. 建议图片分辨率不要过高（建议<4000px）
4. 拼接时会自动调整图片高度（水平拼接）或宽度（垂直拼接）以保持一致
5. 拆分功能适用于之前拼接的图片或规则排列的图片

## 常见问题

**Q: 为什么AI分组很慢？**
A: 首次使用需要下载模型，后续使用会快很多。处理大量图片时需要逐个提取特征，也会较慢。

**Q: 拆分后的图片质量如何？**
A: 使用高质量模式保存（quality=95），最大程度保留原始图片质量。

**Q: 支持哪些图片格式？**
A: 支持常见的图片格式：PNG、JPG、JPEG、BMP、GIF等。

**Q: 可以批量处理多组图片吗？**
A: 可以先使用AI分组功能识别多组图片，然后分别进行拼接。
